# Sentiment-Analysis---Python
A Sentiment Analysis project that visualizes trends in positive, negative, and neutral sentiments over the years using Python, pandas, and matplotlib. The data is sourced from a Kaggle dataset and analyzed to understand sentiment distribution over time.
Features

Reads and processes sentiment data from a CSV file.
Categorizes data into Positive, Negative, and Neutral sentiments.
Aggregates sentiment counts by Year.
Plots sentiment trends using line charts with markers and different colors for clear visualization.
Compatible with Power BI for further analysis and interactive dashboards.

Technologies Used
Python – for data analysis.
pandas – for data manipulation and aggregation.
matplotlib – for visualization of sentiment trends.
Jupyter Notebook – interactive analysis environment.
Power BI – optional for visualizing data in dashboards.

How to Run
Install required Python packages:pip install pandas matplotlib

Place sentiment_analysis.csv in the project folder.
Run the Jupyter Notebook or Python script: jupyter notebook sentiment_analysis.ipynb
